# 博物馆生存指南

这是一个纯前端的网页小游戏项目，核心逻辑与数据均在本仓库内，直接打开 `index.html` 即可体验。

## 项目结构
- `index.html`：入口页面与基础 DOM 结构。
- `styles.css`：全站样式与布局。
- `main.js`：启动逻辑与页面交互的轻量脚本。
- `game.js`：游戏主流程、状态管理、事件与引导逻辑。
- `data.js`：数值配置与文本数据。
- `DEBUG.md`：调试记录或开发备注。

## 本地运行
直接用浏览器打开 `index.html`，无需构建或依赖安装。

## 云端访问
已经部署在yuexin-uvu.github.io，访问网址即可游玩。
